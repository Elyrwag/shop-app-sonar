# shop-app-sonar
Repozytorium dla oczyszczonego z błędów i podatności projektu z zadania 5 wykrytych przez SonarQube Cloud.

Celem jest usunięcie wszystkich błędów typu:

- Bugs,
- Code Smells,
- Vulnerabilities,
- Security Hotspots,
- Code Duplicates.
